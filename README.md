## 🚀 Funcionalidades

Este é um bot multifuncional projetado para animar e administrar seus grupos de WhatsApp.

*   ✅ **Administração de Grupos** (Banir, Promover, Fechar grupo, Anti-link, etc)
*   ✅ **Figurinhas** (Imagens, GIFs, Textos animados)
*   ✅ **Inteligência Artificial**
*   ✅ **Downloads** (TikTok, YouTube, etc)
*   ✅ **Diversão** (Jogos, Brincadeiras)

## 🛠️ Tecnologias

*   Node.js (v22+)
*   Baileys (Conexão WhatsApp)
*   Spider X API (IA e Downloads)
*   Linker API (Canvas e Stickers)
